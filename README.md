## 系统功能

- **账户管理**：创建账户、查看余额、更新账户信息
- **资产管理**：查看和管理资产（股票、期货等）
- **订单管理**：提交限价单、市场单等多种订单类型，支持查询、修改和取消
- **交易执行**：根据订单类型执行交易，更新资产和账户余额
- **历史查询**：查询交易历史记录
- **资金管理**：充值、提现操作，确保账户资金完整性
- **异常处理**：处理资金不足、订单无效等常见异常

## 核心技术要求

- **面向对象**：Account、Asset、Order 等类的设计与实现
- **继承与多态**：Order 基类及其派生类（限价单、市场单）
- **数据结构**：使用队列、堆等结构存储和排序订单
- **并发编程**：处理多用户同时交易，确保系统高效运行
- **文件操作**：持久化存储账户信息和交易历史
- **异常处理**：捕获和处理资金不足、订单错误等异常
